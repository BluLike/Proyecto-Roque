----------------------
Desert Skybox Pack
----------------------

Desert Skybox Pack is a collection of high-resolution skyboxes for Desert Environment.

Contains:
- 7 textures: 6 realistic sky + 1 cartoon sky.
- 7 skybox materials.
- 7 demo scenes.

Texture size:
8192 x 4096 pixels

The demo scenes are located in the folder "\Assets\Desert Skybox Pack\Demo Scenes\".

This project has been tested on unity 2017.4.1/ 2018.4.3/ 2019.2.12.
